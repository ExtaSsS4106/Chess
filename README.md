# Chess Mobile

Мобильное приложение для игры в шахматы на Android.

## 📋 Описание

**Chess Mobile** - это полнофункциональное Android приложение для игры в шахматы. Приложение предоставляет возможность играть с различными противниками, управлять учётной записью и наслаждаться классической игрой в шахматы на мобильном устройстве.

## 🚀 Особенности

- 🎮 Полная реализация правил игры в шахматы
- 👤 Система авторизации и управления аккаунтом
- 🌐 API интеграция для многопользовательской игры
- 📱 Интуитивный интерфейс на Material Design
- ⚡ Оптимизированная производительность

## 📲 Требования

- **Android SDK**: минимум 24 (Android 7.0)
- **Target SDK**: 36 (Android 15)
- **Java**: версия 11+
- **RAM**: минимум 2 GB

## 🔧 Установка и сборка

### Предварительные требования

1. Установите [Android Studio](https://developer.android.com/studio)
2. Установите Java Development Kit (JDK) версии 11 или выше
3. Убедитесь, что Android SDK версии 36 установлен

### Шаги для сборки

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/yourusername/Chess_mobile.git
   cd Chess_mobile
   ```

2. **Откройте проект в Android Studio**
   - Запустите Android Studio
   - Выберите "Open an Existing Project"
   - Выберите папку `Chess_mobile`

3. **Дождитесь синхронизации Gradle**
   - Android Studio автоматически загрузит все зависимости

4. **Соберите проект**
   ```bash
   ./gradlew build
   ```

5. **Запустите приложение**
   - Подключите устройство Android или запустите эмулятор
   - Нажмите "Run" в Android Studio или выполните:
   ```bash
   ./gradlew installDebug
   ```
